Documentation des Routes de l'Application

NB: Aucune requete vers cette api ne sera traiter sans la connexion

GET /api/typesTaches

Récupère la liste des types de tâches.

GET /api/typesTaches/:id

Récupère un type de tâche par ID.

POST /api/typesTaches

Crée un nouveau type de tâche.

PUT /api/typesTaches/:id

Met à jour un type de tâche existant.

DELETE /api/typesTaches/:id

Supprime un type de tâche.

GET /api/taches

Récupère la liste des tâches.

GET /api/filterAdmin/taches

Récupère les tâches filtrées pour les administrateurs.

GET /api/taches/getAllforResp

Récupère toutes les tâches du responsable connecté.

GET /api/filter/taches

Filtre les tâches.

GET /api/search/taches

Recherche des tâches.

GET /api/taches/:id

Récupère une tâche par ID.

POST /api/taches

Ajoute une nouvelle tâche.

PUT /api/taches/:id

Modifie une tâche existante.

DELETE /api/taches/:id

Supprime une tâche.

GET /api/responsables/log

Récupère un responsable par ID.

GET /api/projets

Récupère la liste des projets.

GET /api/statusProjets/:id/projets

Récupère les projets par statut.

GET /api/projets/:id

Récupère un projet par ID.

POST /api/projets

Ajoute un nouveau projet.

GET /api/search/projets

Recherche un projet.

GET /api/filter/projets

Filtre les projets.

PUT /api/projets/:id

Met à jour un projet existant.

DELETE /api/projets/:id

Supprime un projet.

POST /api/profiles/auth

Route de connexion.

PUT /api/profiles/auth/updatePassword

Met à jour le mot de passe d'un profil.

POST /api/profiles/logout

Déconnexion d'un profil.

GET /api/profiles/log

Récupère un profil par ID.

GET /api/observations

Récupère la liste des observations.

GET /api/projets/:id/observations

Récupère toutes les observations d'un projet.

GET /api/observations/:id

Récupère une observation par ID.

POST /api/observations

Ajoute une nouvelle observation.

PUT /api/observations/:id

Met à jour une observation existante.

DELETE /api/observations/:id

Supprime une observation.
